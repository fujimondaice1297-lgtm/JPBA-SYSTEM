# 公式選手ページからのタイトル明細復元（2026-07-14）

## 目的

現行JPBA選手ページの保有タイトル数と、リニューアルDBの大会名付きタイトル明細を一致させる。
年度別出場大会の `1位` 行を根拠にするが、選抜・予選などタイトルに数えない1位行や、古い年度ページの例外を集計上限で検出し、推測登録しない。

## 実装

- `JpbaOfficialPlayerTitleHistoryService` を追加し、選手ページから年度URLと1位行を抽出するようにした
- `JpbaOfficialHallOfFameTitleService` を追加し、日本プロボウリング殿堂の `MainTitle` 一覧を解析するようにした
- `jpba:import-official-player-title-history` を追加した
  - 既定はドライラン
  - `--force --promote` のときだけ候補保存と本登録を行う
  - 保有数を超える場合はカテゴリー単位で登録をブロックする
  - `--license`、`--min-year`、`--max-year`、`--limit`、`--offset` で再開範囲を限定できる
- 公式ページHTMLは `storage/app/private` の非公開キャッシュへ保存し、不完全取得時は処理を失敗させる
- 殿堂ページの同一タイトル重複を除去し、括弧だけの行は直前の大会名へ結合するようにした
- 殿堂掲載後にタイトルが増えた選手は、殿堂タイトルと掲載最終年より後の年度別1位を合成するようにした
- 大会名が選抜・予選会・予選大会・予選ラウンドで終わる1位行は、公式タイトル候補から除外するようにした
