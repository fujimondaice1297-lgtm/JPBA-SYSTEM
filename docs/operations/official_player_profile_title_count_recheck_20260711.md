# 現行JPBAサイト 個人成績カウント再チェック（2026-07-11）

## 背景

藤永北斗プロ（M00001443）の管理画面で、現行JPBAサイトでは公式優勝5回・シーズントライアル優勝3回であるにもかかわらず、リニューアル側の表示が0になっていた。

原因は、`pro_bowlers.official_win_count` には公式優勝回数が入っていた一方で、管理画面が未復元の `pro_bowler_titles` 明細件数を優先していたこと。また、現行サイトの優勝回数欄に含まれるシーズントライアル注記を保存するカラムがなかったこと。

## 対応内容

- `pro_bowlers` に `season_trial_win_count` を追加した。
- 現行JPBAサイトの選手詳細ページから、優勝回数欄の括弧内にあるシーズントライアル優勝回数を読み取るようにした。
- 管理画面のタイトル情報は、タイトル明細が未復元でも `official_win_count` / `season_trial_win_count` の集計値を表示するようにした。
- 公開プロフィールも、公式タイトル数とシーズントライアル優勝回数の集計値を表示するようにした。
- 選手一覧・管理検索のタイトル数は、タイトル明細件数と公式サイト集計値の大きい方を使うようにした。
