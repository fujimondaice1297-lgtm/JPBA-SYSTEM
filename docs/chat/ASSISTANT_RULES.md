# ChatGPT（DB設計支援）への依頼ルール

## 必須の出力形式（毎回）

1. **どのファイルに**作る/編集するか（パスを明記）
2. **VSCodeでの作業手順**（右クリック→新規ファイル、どこを開く、保存など）
3. **全文コピペ可能**な内容（差し替え対象ファイルは全文提示）

## DB設計変更のときの必須セット

DBの変更は必ず以下をセットで更新する：

- migrations
- docs/db/data_dictionary.md（正本）
- docs/db/ER.dbml（自動生成結果。手編集しない）

## 禁止事項

- 不明な前提を決め打ちしない（根拠がない参照は refs_skipped / ADR に残す）
- ER.dbml を直接編集しない（辞書側を修正して再生成）
